# Student Grade Calculator (Kotlin & Dart)

This project is a multi-platform academic utility designed to demonstrate expertise in both **Kotlin (Native Android)** and **Dart (Flutter)**.

## Project Evolution

### Milestone 1: Functional Programming
- **Focus:** Demonstrating core functional concepts.
- **Implementation:** Usage of Lambdas, Higher-Order Functions (HOFs), and efficient collection processing to calculate grades and filter student data.
- **Status:** Completed in both Kotlin and Dart.

### Milestone 3: Object-Oriented Domain Model
- **Focus:** Refinement of the data model using advanced OOP principles.
- **Requirements:**
  - Inheritance from an abstract base class (`AcademicEntity`).
  - Interface implementation (`GradeReportable`).
  - Polymorphic behavior when processing collections of different entities (Students, Instructors).
  - Use of Data Classes (Kotlin) and robust class structures with `copyWith` (Dart).
- **Status:** Completed in both Kotlin and Dart.

---

## Technical Features (Kotlin Version)
- **MVVM Architecture:** Clean separation of UI, Logic, and Data.
- **Jetpack Compose:** Modern declarative UI implementation.
- **Room Database:** Persistent local storage for student records.

## Technical Features (Dart/Flutter Version)
- **Cross-Platform:** Shared logic across iOS and Android.
- **Provider Pattern:** Efficient state management.
- **Equivalent OOP:** Matching class hierarchies to the Kotlin implementation.
